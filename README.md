# Counter App

A simple Counter Application built using **HTML, CSS, and JavaScript**.

## 🚀 Features

* Increase counter value
* Decrease counter value
* Reset counter
* Dynamic color change based on count value

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 📂 Project Structure

counter-app
│── index.html
│── style.css
│── script.js

## 🎯 How It Works

* Click **+** button to increase the count
* Click **-** button to decrease the count
* Click **Reset** to set the counter back to **0**
* Counter color changes depending on the value
